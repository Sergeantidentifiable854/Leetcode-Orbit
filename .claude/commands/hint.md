用户在做题过程中请求提示。请严格按照以下流程执行：

## 第一步：确定当前题目

读取 `workspace/.current_meta.json` 获取当前题目信息。
如果文件不存在，告诉用户"当前没有正在做的题目，先用 /practice 开始一道题吧"。

## 第二步：判断提示级别

检查本次对话中用户已经获得过几次提示（通过对话历史判断）：

- 第 1 次请求 → Level 1
- 第 2 次请求 → Level 2
- 第 3 次请求 → Level 3
- 用户说"放弃/看答案/give up/show answer" → Level 4

用户可以直接指定级别（如 "hint 3", "给我 level 2 提示"），此时直接跳到指定级别。

## 第三步：按级别给出提示

### Level 1 — 方向引导（无剧透）
- 只告诉用户该用什么**类型**的方法/数据结构
- 用提问的方式引导思考
- ❌ 不提及具体算法名称，不给任何代码
- 格式：
```
💡 Level 1 提示 — 方向引导

{1-2句引导性问题，指向正确的方法类型}

想继续思考？还是需要更多提示？说 "hint" 可以获取下一级帮助。
```

### Level 2 — 思路框架（轻微剧透）
- 给出分步骤的思考框架
- 可以提及关键数据结构名称、描述核心逻辑
- ❌ 不写任何代码或伪代码
- 格式：
```
💡 Level 2 提示 — 思路框架

{分步骤的思考引导，3-5步}

需要代码骨架？说 "hint" 获取 Level 3 提示。
```

### Level 3 — 伪代码/填空（显著剧透）
- 提供代码骨架，关键部分用 `______` 留空
- ❌ 不给完整可运行的答案
- 格式：
```
💡 Level 3 提示 — 代码骨架

```python
{代码骨架，关键逻辑处留空用 ______ 代替}
```

试着把 ______ 填上，如果实在不行，说 "看答案" 我会给你完整解答。
```

### Level 4 — 完整答案 + 讲解
1. 从 `references/index.json` 找到当前题的参考答案位置
2. 读取对应区段的参考答案
3. 逐行讲解代码
4. 给出核心思想总结和复杂度分析
5. 鼓励用户
- 格式：
```
📖 Level 4 — 完整答案

{参考答案代码 + 逐行讲解}

🎯 核心思想: {一句话总结}
⏱ 复杂度: 时间 {O(?)} / 空间 {O(?)}

看答案不丢人，重要的是理解了下次能自己写出来。
我会在几天后再出这道题给你，到时候试试不看提示独立完成！
```

## 第四步：记录提示使用

在 `workspace/.current_meta.json` 中追加提示记录：
```json
{
  "hint_used": true,
  "max_hint_level": 2,
  "hint_weaknesses": ["链表指针操作记忆模糊"]
}
```

### hint_weaknesses 写入规则

每次给出提示时，根据用户的求助内容判断薄弱点，用简短关键词写入 `hint_weaknesses` 数组：

- 用户说"不记得怎么做" → 对应知识点 + "记忆模糊"（如 "链表指针操作记忆模糊"）
- 用户说"不会/完全没思路" → 对应知识点 + "不熟练"（如 "DP状态转移不熟练"）
- 用户方向错误（如该用哈希却想暴力） → "未识别最优算法"
- 用户知道方向但细节卡住 → 具体细节（如 "递归终止条件不清晰"）

多次提示时，`hint_weaknesses` 累积追加，不覆盖之前的。`max_hint_level` 取最大值。

### 权重影响

这个信息会在 review 阶段被读取，影响权重计算：

| 提示级别 | 全通过无薄弱点 | 全通过有薄弱点 | 未全部通过 |
|---------|--------------|--------------|----------|
| 无提示   | `pw * 0.6`   | `pw * 0.8`   | 正常提升  |
| L1      | `pw * 0.8`   | `pw * 0.9`   | 正常提升  |
| L2      | 维持不变      | 维持不变      | 正常提升  |
| L3      | 维持不变      | 维持不变      | 正常提升  |
| L4      | `pw * 1.2`   | `pw * 1.2`   | `pw * 1.2` |

## 注意事项
- 具体技术问题（"这里用 defaultdict 还是 dict？"）和 debug 请求（"为什么这个用例出错？"）直接回答，不消耗提示级别
- 只有"我不知道怎么做"类型的求助才进入四级递进
- Level 4 读取参考答案时只读当前题的区段，读完后不保留在上下文中
